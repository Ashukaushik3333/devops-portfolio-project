# Repos for this Project

This folder links the two main repos used:

- **flask-app** — Application + CI
  - Repo: https://github.com/Ashukaushik3333/flask-app
  - Purpose: Flask app, Dockerfile, GitHub Actions that build & push image.

- **infra-k8s** — Helm chart + CD
  - Repo: https://github.com/Ashukaushik3333/infra-k8s
  - Purpose: Helm chart, ServiceMonitor, GitHub Actions to deploy the chart.

Clone commands:
```bash
git clone https://github.com/Ashukaushik3333/flask-app.git
git clone https://github.com/Ashukaushik3333/infra-k8s.git
```
